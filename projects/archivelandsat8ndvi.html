Landsat 8 ndvi

Landsat 8 ndvi is a demo tool for using Landsat data on AWS

  1. Use the draw tool (top right) to pin on an area of interest
  2. Visualize NDVI value for all Landsat 8 images

Filter is set to select images with less than 20% of cloud coverage

Landsat 8 images before 2015 may not be available for processing

NDVI is calculated from TOA (Top Of Atmosphere) reflectance values (no atmopheric correction)

Search results are powered by RemotePixel Landsat API
