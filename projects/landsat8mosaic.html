Landsat 8 Mosaic

"Landsat 8 Mosaic" is a tool to visualize and create quick Landsat 8 satellite imagery mosaic.

This is not science! mosaic are created from jpeg preview

  1. Draw an area of interest
  2. Click on one scenes row-path combination available to select each landsat scene or click on 'show latest'/'complete'
  3. Click on an image to replace it
  4. Request mosaic creation using api.remotepixel.ca (Email requested but not saved by remotepixel)
  5. Get selected landsat ids to do your own processing with Landsat-util
  6. Filters at the top of the map can be used to select a new date range, cloud cover percentage or season

Search results are powered by RemotePixel Landsat API and images from USGS Earth Explorer.

Mosaic creation request powered by RemotePixel API.
