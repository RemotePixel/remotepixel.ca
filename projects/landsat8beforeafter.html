Landsat 8 Before/After

Landsat 8 Before/After is a tool to pre-visualize earth changes using open Landsat 8 satellite imagery.

  1. Dragg the THE MAP to an area of interest
  2. Click on one scenes row-path combination available
  3. Select images you want to compare
  4. Filters can be used to select a new date range, and cloud cover percentage or seasons

Search results are powered by Landsat 8 Metadata API and images from USGS Earth Explorer.
