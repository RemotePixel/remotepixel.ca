MODIS-VIIRS GIF

Modis-VIIRS Gif is a tool to ease GIF creation of NASA MODIS and VIIRS imagery. Within a few click users can make GIF annimation (max 25 img) using imagery archive up to 2012.

NASA Imagery from EOSDIS Global Imagery Browse Services (GIBS) using awesome Mapbox Mapbox GL to render images.
