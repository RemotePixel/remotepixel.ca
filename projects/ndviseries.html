NDVI Series

Ndvi Serie is a demo tool using AWS Lambda to process Landsat/Sentinel data on AWS

How To:

  1. Use the draw tool (top right) to select an area of interest (maximun of 1000kmÂ² for polygons)
  2. Visualize NDVI value for all Landsat 8/Sentinel2 images

NDVI is calculated from TOA (Top Of Atmosphere) reflectance values (no atmopheric correction)
