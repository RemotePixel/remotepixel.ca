Landsat 8 Evolution

Landsat 8 Evolution is a tool to visualize earth changes using open Landsat 8 satellite imagery.

  1. Dragg the THE MAP to an area of interest
  2. Click on one scenes row-path combination available
  3. Use the time slider on the bottom left to pass over each available scene
  4. Request animated GIF creation using api.remotepixel.ca (Email requested but not saved by remotepixel)
  5. Filters at the top of the map can be used to select a new date range, and cloud cover percentage

Search results are powered by RemotePixel Landsat API and images from USGS Earth Explorer.
