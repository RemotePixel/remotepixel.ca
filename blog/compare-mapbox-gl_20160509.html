Using Mapbox-GL to Improve MODIS-VIIRS Imagery comparison

After my first taste of GL in poster.remotepixel.ca (blog) I decided to update some of my previous projects.

Last week I used a lot the MODIS-VIIRS before/after map project to see the evolution of the Fort McMurray fire and I feeled this project needed some updates/improvements. Using Mapbox-gl able a better navigation in the map and using their Compare Tool make the project really robust on all browsers.

So here it is: MODIS-VIIRS map compare

I hope you'll like the new version. if you see anything that could be improved please contact me contact@remotepixel.ca
